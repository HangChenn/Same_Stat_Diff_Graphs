This is the codes and data generated for Same Stat, Diff Graph project.

Noted the data in this folder is not all data we generated.
Some files are huge and fairly easy to generated by using the code we provided. 

-------------------
Structure

the project is seperate to three parts,

the first part is the data generating tool, we used it to generate ground-truth data from a total set of non-isomorphic graph.
this tool also able to generate graph set by given the generator name

the second part is static visulazition tool, which include parallel coordinates plot, the upper-trinagle matrix and violin plot.

the third part is the exploring tool you see in the website, which allow you to fix several satistic and vary another one.

For each part, we provide an individual readme file
